# 金融サイクルと景気循環

金融サイクルは、金利、信用、資産価格、景気が相互に影響しながら循環する構造。典型的には、中央銀行の引き締めと緩和、銀行貸出、企業収益、不動産価格、債券価格、株価が時間差を持って動く。

## 1. 典型的な資産価格の流れ

ユーザーが挙げた「金利が上がり、不動産/債券が下がり、株が上がり、金利が下がり、不動産/債券が上がり、株が下がる」という流れは、景気サイクルの中でよく観察されるが、常に同じ順序ではない。大まかには次のように整理できる。

| 局面 | 金利 | 債券 | 不動産 | 株式 | 主な背景 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **回復初期** | 低位 | 高止まり | 底打ち | 上昇開始 | 金融緩和、景気底打ち、リスク許容度回復 |
| **拡大中期** | 緩やかに上昇 | 下落しやすい | 上昇 | 上昇 | 企業利益改善、信用拡大、リスクオン |
| **過熱期** | 上昇 | 下落 | 上昇鈍化/過熱 | 上昇だが不安定 | インフレ、引き締め、レバレッジ拡大 |
| **減速期** | 高止まり後に低下 | 上昇しやすい | 下落/調整 | 下落 | 利益減速、信用不安、利下げ織り込み |
| **後退期** | 低下 | 上昇 | 下落 | 下落後に底打ち | 景気後退、信用収縮、政策対応 |

重要なのは「金利上昇 = 株安」と単純化しないこと。景気が強く利益成長が金利上昇を上回る局面では、金利上昇と株高が同時に起きる。一方、インフレや財政不安で長期金利が上がる局面では、株式と債券が同時に下がることもある。

## 2. 金利と債券・不動産・株式の関係

### 債券
- 金利が上がると既存債券価格は下がる。
- 金利が下がると既存債券価格は上がる。
- 長期債ほど金利変化に敏感で、デュレーションリスクが大きい。

### 不動産
- 金利上昇は住宅ローン、REITの借入コスト、キャップレート上昇を通じて不動産価格に逆風。
- ただし景気拡大と賃料上昇が強ければ、不動産価格は金利上昇中でも上がることがある。
- 金利低下は不動産価格に追い風だが、景気後退で空室率や賃料が悪化すれば相殺される。

### 株式
- 金利上昇は割引率上昇としてグロース株に逆風。
- 景気拡大による金利上昇なら、景気敏感株や金融株には追い風になり得る。
- 金利低下はバリュエーションには追い風だが、景気後退による利下げなら利益悪化で株価は下がり得る。

## 3. 中央銀行と金融サイクル

| 政策局面 | 目的 | 市場への典型的影響 |
| :--- | :--- | :--- |
| **利上げ開始** | インフレ抑制、過熱抑制 | 短期金利上昇、イールドカーブフラット化 |
| **利上げ継続** | 需要抑制 | 不動産・長期債・グロース株に逆風 |
| **金利高止まり** | インフレ再燃防止 | 借換負担、信用スプレッド拡大リスク |
| **利下げ開始** | 景気下支え | 債券高、通貨安、株式は景気次第 |
| **量的緩和/流動性供給** | 金融ストレス緩和 | リスク資産の下支え、信用市場改善 |

## 4. 先進国と新興国の循環差

### 先進国
- 金融政策の信頼性が高く、長期金利はインフレ期待とタームプレミアムに反応しやすい。
- 米国は世界のドル流動性の中心で、FRBの政策は他国の資本フローに影響する。
- 欧州はエネルギー価格、財政ルール、銀行信用の影響を受けやすい。

### 新興国
- 米金利とドル指数の影響が大きい。
- 米金利上昇・ドル高は、資本流出、通貨安、外貨建て債務負担増につながりやすい。
- コモディティ輸出国は資源価格サイクルに左右されやすい。
- インフレ抑制のため、先進国より早く利上げすることがある。

| 環境 | 先進国 | 新興国 |
| :--- | :--- | :--- |
| **米金利上昇・ドル高** | グロース株に逆風、金融環境引き締め | 通貨安、資本流出、外貨債務負担 |
| **米金利低下・ドル安** | リスク資産に追い風になりやすい | 資本流入、通貨高、株式・債券に追い風 |
| **資源高** | 消費国にはインフレ圧力 | 資源輸出国には交易条件改善 |
| **世界景気減速** | 金利低下と利益減速が同時進行 | 輸出減、資本流出、信用不安 |

## 5. 日本の景気循環の特徴

日本は米国や新興国と比べて、インフレ率、賃金、人口動態、金融政策、為替の影響が大きく異なる。

- **低金利期間が長い**: 国内金利の変化より、米金利とドル円の影響が大きくなりやすい。
- **輸出企業の比重**: 円安は輸出企業の利益押し上げ要因、円高は逆風。
- **内需・不動産**: 国内金利上昇は不動産、REIT、借入依存企業に逆風。
- **海外投資家主導**: 日本株の需給は米株、米金利、ドル円、海外投資家フローに左右されやすい。
- **日銀政策**: YCC、マイナス金利解除、国債買入、ETF保有など、政策変更が市場構造に影響する。

## 6. サイクルのズレを見る

米国、新興国、日本は同じタイミングで景気循環が進むとは限らない。

| 組み合わせ | 典型例 | 市場への影響 |
| :--- | :--- | :--- |
| **米国拡大・日本回復** | 米景気強く、円安進行 | 日本の輸出株に追い風 |
| **米国減速・日本遅行回復** | 米金利低下、円高圧力 | 日本株は外需株中心に重くなりやすい |
| **米国引き締め・新興国脆弱** | ドル高、資本流出 | 新興国株・通貨に逆風 |
| **中国減速・日本外需悪化** | 資源・機械・素材需要低下 | 日本の景気敏感株に逆風 |
| **世界同時緩和** | 金利低下、流動性改善 | 債券、不動産、グロース株に追い風になりやすい |

## 7. 実務チェック

- 金利上昇が「景気の強さ」なのか「インフレ/財政不安」なのかを分ける。
- 金利低下が「金融緩和期待」なのか「景気後退懸念」なのかを分ける。
- 債券高と株高が同時に起きている場合、流動性相場か、利下げ期待かを確認する。
- 株安と債券安が同時に起きている場合、インフレ再燃、財政不安、実質金利上昇を疑う。
- 日本株では米金利、ドル円、海外投資家売買、日銀政策、中国景気を同時に見る。
