# 投資スタイル (Momentum / Quant / Value)

投資スタイルは、銘柄選択やポートフォリオ構築の考え方を分類する枠組み。実務では単独で使うより、バリュー、モメンタム、クオリティ、低ボラ、サイズ、需給などを組み合わせて使うことが多い。

## 1. モメンタム投資

モメンタム投資は、過去に強かった資産や銘柄が短中期的にさらに強くなりやすい、という経験則に基づく投資手法。

### 主な考え方
- **価格モメンタム**: 過去3か月、6か月、12か月などのリターンが高い銘柄を買う。
- **業績モメンタム**: EPS上方修正、売上成長、利益率改善など、ファンダメンタルズの改善が続く銘柄を買う。
- **需給モメンタム**: 出来高増加、海外投資家買い、空売り買い戻しなどを確認する。

### よく使う指標
| 指標 | 内容 |
| :--- | :--- |
| **過去リターン** | 1か月、3か月、6か月、12か月の騰落率 |
| **相対リターン** | TOPIXや業種指数に対する超過リターン |
| **移動平均乖離率** | 価格が中長期トレンドに対してどれだけ強いか |
| **業績修正** | アナリスト予想や会社予想の上方修正 |
| **出来高変化** | 新しい資金流入やテーマ化を確認 |

### 強み
- トレンドが続く局面で利益を伸ばしやすい。
- ファンダメンタル改善や需給変化を早く取り込める。
- ルール化しやすく、クオンツ戦略と相性が良い。

### 弱み
- 急なリバーサルに弱い。
- 高値掴みになりやすい。
- 市場全体のリスクオフ時に一斉に巻き戻されることがある。

## 2. バリュー投資

バリュー投資は、企業価値に対して市場価格が割安な銘柄を買い、価格が価値に近づくのを待つ投資手法。

### 主な考え方
- **低PER**: 利益に対して株価が安い。
- **低PBR**: 純資産に対して株価が安い。
- **高配当利回り**: 株主還元に対して価格が安い。
- **資産価値**: 現金、不動産、政策保有株などの資産価値に注目する。
- **平均回帰**: 一時的に嫌われた銘柄が正常評価に戻ることを狙う。

### よく使う指標
| 指標 | 内容 | 注意点 |
| :--- | :--- | :--- |
| **PER** | 株価 / EPS | 景気循環株では利益ピーク時に低PERになりやすい |
| **PBR** | 株価 / BPS | 資本効率が低い企業は低PBRが続くことがある |
| **EV/EBITDA** | 企業価値 / EBITDA | 負債を含めた比較に向く |
| **配当利回り** | 配当 / 株価 | 減配リスクを確認する |
| **FCF利回り** | フリーCF / 時価総額 | 会計利益より現金創出力を見やすい |

### 強み
- 長期的にリスクプレミアムが観測されやすい。
- 下値余地を企業価値で説明しやすい。
- 株主還元、資本効率改善、アクティビスト、東証改革と相性が良い。

### 弱み
- 割安に見えても構造的に悪い企業、いわゆるバリュートラップがある。
- 成長株優位の局面では長期間劣後しやすい。
- 会計上の利益や簿価が実態価値を表さないことがある。

## 3. クオンツ投資

クオンツ投資は、データ、統計、ルール、モデルを使って投資判断を行う手法。人間の裁量を減らし、再現可能なルールに落とし込むことを重視する。

### 主なアプローチ
- **ファクター投資**: バリュー、モメンタム、クオリティ、低ボラ、サイズなどのファクターに分散投資する。
- **統計的裁定**: ペアトレード、平均回帰、相関崩れを狙う。
- **機械学習**: 多数の特徴量からリターンやリスクを予測する。
- **イベントドリブン**: 決算、上方修正、自社株買い、指数採用などのイベント反応をモデル化する。
- **リスクパリティ**: 資産ごとのリスク寄与度を均等化する。

### データ設計
| 観点 | 内容 |
| :--- | :--- |
| **特徴量** | 価格、出来高、財務、需給、マクロ、ニュース |
| **目的変数** | 将来リターン、超過リターン、ボラティリティ、ドローダウン |
| **ユニバース** | TOPIX、プライム市場、流動性上位、業種中立など |
| **リバランス** | 日次、週次、月次、決算後など |
| **制約** | 売買代金、セクター、銘柄上限、回転率、コスト |

### 強み
- ルールが明確で検証しやすい。
- 多銘柄に分散しやすい。
- 感情による判断ミスを減らせる。

### 弱み
- 過剰最適化しやすい。
- データ欠損や先読みバイアスに弱い。
- 市場構造が変わると過去の優位性が消えることがある。
- 取引コストと流動性制約で机上の成績が消えやすい。

## 4. 組み合わせ方

| 組み合わせ | 狙い | 注意点 |
| :--- | :--- | :--- |
| **Value + Momentum** | 割安で、かつ改善が始まった銘柄を狙う | 反転初期を逃すことがある |
| **Quality + Value** | 安いだけでなく収益性・財務健全性も見る | 高Quality銘柄は割安になりにくい |
| **Momentum + Risk Control** | トレンドを取りつつ急落を抑える | 損切りが多いと回転率が上がる |
| **Quant + Fundamental** | 定量スクリーニング後に個別分析する | 裁量でルールが崩れないようにする |

## 5. 日本株での実務ポイント

- 小型株はファクター効果が出やすいが、流動性制約が強い。
- バリュー投資ではPBR 1倍割れ、資本効率、株主還元、政策保有株の削減を確認する。
- モメンタム投資では決算直後の上方修正、業種テーマ、海外投資家の買いを確認する。
- クオンツ投資では上場廃止銘柄、売買停止、分割・併合、配当、決算発表時刻を正しく扱う。
- どのスタイルも、単独の指標ではなく、リスク、流動性、コスト、レジームを合わせて評価する。
