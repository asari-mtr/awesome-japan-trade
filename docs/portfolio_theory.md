# ポートフォリオ理論 (Portfolio Theory)

複数の資産を組み合わせることで、リスクを抑えつつリターンを最大化するための理論。

## 1. 現代ポートフォリオ理論 (MPT: Modern Portfolio Theory)
ハリー・マーコウィッツによって提唱された。
- **分散投資**: 相関の低い資産を組み合わせることで、リターンを維持したままポートフォリオ全体のリスク（標準偏差）を低減できる。
- **平均・分散アプローチ**: リターン（期待値）とリスク（分散）の2軸で投資対象を評価する。

## 2. 効率的フロンティア (Efficient Frontier)
あるリスクに対して最大のリターン、またはあるリターンに対して最小のリスクを実現するポートフォリオの集合。
- 投資家はこのフロンティア線上にあるポートフォリオを選択することが合理的とされる。

## 3. CAPM (Capital Asset Pricing Model)
個別資産の期待リターンを市場全体のリスクとの関係性で説明するモデル。
- **公式**: $E[R_i] = R_f + \beta_i (E[R_m] - R_f)$
    - $R_f$: リスクフリーレート
    - $\beta$: 市場に対する感応度
    - $E[R_m] - R_f$: 市場リスクプレミアム
- **ベータ($\beta$)**: 市場全体の変動に対して、その資産がどれだけ動くかを示す指標。

## 4. シャープ・レシオ (Sharpe Ratio)
リスク1単位あたりに得られた超過リターンを示す。
- **計算式**: (ポートフォリオのリターン - 無リスク金利) / 標準偏差
- 値が高いほど、効率の良い運用ができていると判断される。

## 5. ファクター・モデル
資産リターンを複数の要因（サイズ、バリュー、モメンタム等）で説明する手法。
- Fama-Frenchの3ファクター・モデルなどが有名。
