# 市場構造と高度な分析手法 (Market Structure & Advanced Analysis)

日本市場、特に先物・オプション市場においてプロの投資家が注目する高度な分析概念です。

## 1. 建玉構造 (Open Interest Structure)
建玉（未決済残高）の分布は、将来の買い・売りの「潜在的な壁」や「加速要因」を示唆します。
- **最大マックスペイン (Max Pain)**: オプションの買い手の損失が最大、売り手の利益が最大となる価格帯。SQ（特別清算指数）に向けて価格が収束する傾向があるという仮説。
- **権利行使価格別の建玉分布**: 特定の価格帯（例：40,000円）に建玉が集中している場合、その価格が強い抵抗線または支持線として機能することがあります。
- **注意**: 建玉は買い手・売り手のどちらが投機筋か、ヘッジ目的かを直接示さない。単純な建玉の多寡だけで支持線・抵抗線と断定しない。

## 2. ガンマエクスポージャー (GEX)
マーケットメイカー（オプションの売り手）のヘッジ行動が原資産（日経平均）に与える影響を数値化した指標。
- **ポジティブ・ガンマ (Long Gamma)**: 相場のボラティリティを抑制する方向にヘッジが入る状態。
- **ネガティブ・ガンマ (Short Gamma)**: 相場が動いた方向にさらに加速させるヘッジ（追随売り・追随買い）が入りやすく、ボラティリティが拡大する状態。
- **ガンマ・フリップ (Gamma Flip)**: GEXがゼロになる価格水準（ゼロガンマ水準）。この水準を境にマーケットメイカーのヘッジ行動が反転します。
  - **Gamma flip より上**: マーケットメイカーはポジティブガンマ状態 → 逆張りヘッジ（上昇時に売り、下落時に買い）→ 相場を安定させる
  - **Gamma flip より下**: マーケットメイカーはネガティブガンマ状態 → 追随ヘッジ（下落時にさらに売り）→ 相場の下落が加速しやすい
  - **上抜けの場合**: Gamma flip を上抜けるとネガティブ→ポジティブへ転換し、急騰が落ち着きやすい
- **Gamma Wall**: GEXが特に大きなプラス値を示す価格帯。マーケットメイカーの逆張りヘッジが集中し、強い価格吸着効果（磁石効果）が働きやすい。Max Painと近い概念だが、GEXの高さで測る点が異なる。
- **Call Wall**: Gamma Wall と同義で使われることが多い。現在値より上に位置する大きなプラスGEX帯を指し、上昇を抑制する天井として機能しやすい。
- **Put Wall**: 現在値より下に位置する大きなマイナスGEX帯。ネガティブガンマが集中しており、価格が下抜けるとMMの追随売りが加速しやすい。下方向のGamma flipと組み合わせて見ると、下落リスクの目安になる。
- **Zero Gamma**: Gamma flip と同義で使われることが多い。GEXがゼロを横切る価格水準を指す。

### Gamma Flip 活用時の注意
- GEXの符号・大きさはMMが「オプション売り越し」という仮定に依存するため、観測できない情報を含む。
- 日次でGamma flipの水準が変化するため、建玉が更新されるSQ後や大きな市場変動後は再計算が必要。
- Gamma flip を抜けたからといって必ず急落・急騰するわけではなく、流動性やマクロイベントとの組み合わせで判断する。

### ガンマ・スクイーズ (Gamma Squeeze)
価格が急上昇し、ショートガンマ（またはショートコール）ポジションを持つ参加者が強制的にデルタヘッジ（原資産の買い）を行うことで、上昇がさらに加速する現象。

- **発生メカニズム**: コールのショートやネガティブガンマのMMが、価格上昇でデルタが増加した分だけ原資産を買い増さざるを得ない → 追随買いで価格がさらに上昇
- **Gamma flip との関係**: Gamma flip を下から上に突き抜ける局面で発生しやすい。突き抜け直後はネガティブガンマ帯からポジティブガンマ帯への移行で売り圧力も増すため、Squeezeは一時的になりやすい。
- **短絡的な解釈の注意**: Gamma Squeeze は原因ではなく増幅器。背後にある需給・マクロ要因を確認する。

### GEX推定時の前提
- **符号の推定**: 公開データだけでは各参加者のロング・ショートを完全には特定できない。マーケットメイカーがオプション売り越しという仮定を置く場合、その仮定を明記する。
- **原資産価格**: 日経平均現物、先物、ミニ先物で価格が異なるため、どの価格を使ったかを固定する。
- **満期別集計**: 期近オプションのガンマはSQ接近で急増する。全限月合算だけでなく、期近・翌限月を分ける。
- **デルタヘッジの実行可能性**: 理論上のヘッジ需要と実際の売買は一致しない。流動性、取引時間、リスク制限の影響を受ける。

### DEX / VEX / Charm
GEXだけでは、方向性、IV変化、時間経過によるヘッジ需要を十分に説明できない。関連するエクスポージャーを分けて見る。

| 指標 | 意味 | 主な用途 |
| :--- | :--- | :--- |
| **DEX (Delta Exposure)** | オプション建玉のデルタ換算エクスポージャー | 市場全体が上方向・下方向どちらに偏っているかを見る |
| **GEX (Gamma Exposure)** | 原資産価格変化に対するデルタ変化の集計 | ヘッジ売買が相場を抑制するか、加速するかを見る |
| **VEX (Vega Exposure)** | IV変化に対する価値変化の集計 | IV上昇・低下時の需給や損益感応度を見る |
| **Charm Exposure** | 時間経過に伴うデルタ変化の集計 | SQ接近時や週末前後のデルタヘッジ需要を見る |

### Gamma / Charm Curve
価格帯ごとにGEXやCharmを再計算した曲線。現在価格だけでなく、相場がどの水準に動くとヘッジ需要が変化するかを確認する。

- **Gamma curve**: 原資産価格ごとのGEX。Gamma wall、zero gamma、gamma flipの推定に使う。
- **Charm curve**: 原資産価格ごとのCharm。時間経過だけでどの価格帯に買い・売りヘッジが出やすいかを見る。
- **Vanna curve**: 原資産価格ごとのVannaエクスポージャー。IV変化が起きたときにどの価格帯でデルタヘッジが誘発されるかを確認する。
- **Vanna / VEXとの併用**: 下落時にIVが上昇する市場では、価格変化とIV変化が同時に起きるため、GEX単独ではヘッジ需要を過小評価しやすい。
- **注意**: これらは建玉、IV、モデル、マーケットメイカーの符号仮定に強く依存する。正確な予測ではなく、需給の地図として使う。

### ヴァナ・フロー (Vanna Flow)
IVの変化がデルタヘッジ需要を引き起こす連鎖。GEXが原資産価格の変化に対するヘッジ需要を表すのに対し、Vanna Flowは**IV変化に伴うデルタ変化（ヘッジ需要）**を指す。

$$\text{Vanna} = \frac{\partial^2 V}{\partial S \, \partial \sigma} = \frac{\partial \Delta}{\partial \sigma}$$

| 状況 | IVの動き | Vannaの影響 |
| :--- | :--- | :--- |
| 下落局面 | IV上昇 | デルタが減少 → MMがさらに売りヘッジ → 下落を加速 |
| 反発局面（IV低下） | IV低下 | デルタが増加 → MMが買い戻しヘッジ → 反発を増幅 |
| IV crush（決算後など） | IV急低下 | コール保有者のデルタが増加 → MMの売り → 相場の上げが鈍る |

- **GEXとの違い**: GEXは原資産価格だけで動くが、Vanna FlowはIVが変化するたびに発動する。IV急変時はGEXよりもVanna Flowが主役になりやすい。
- **日本市場での注意**: 日経平均は下落時にIVが急騰しやすい（負のスキュー）ため、下落局面のVanna Flowは特に強く働く傾向がある。

## 3. 裁定取引 (Arbitrage) とプログラム売買
現物株と先物の価格差を利用した取引。
- **裁定買い残**: 先物を売って現物を買っている状態。この残高が積み上がると、将来的に現物を売って先物を買い戻す「裁定解消売り」が発生し、株価の下げ要因となります。
- **逆裁定（裁定売り残）**: 将来的な買い戻し要因となります。
- **ベーシス**: 先物価格と現物指数の差。金利、配当、需給、残存期間で変化する。
- **注意**: 裁定残は将来の需給圧力の候補であり、解消タイミングを直接予測する指標ではない。

## 4. 投資家部門別売買動向
誰が日本株を動かしているかを把握するためのデータ。
- **海外投資家**: 日本株の売買シェアの約7割を占め、トレンドを形成。
- **個人投資家**: 逆張り（株価下落時に買い）の傾向が強い。
- **信託銀行**: 年金基金等のリバランス売買。
- **自己（証券会社）**: 裁定取引やマーケットメイク。

### 読み方の注意
- 週次データは公表までラグがあるため、リアルタイム売買判断ではなく、需給トレンドの確認に使う。
- 現物、先物、オプションを別々に見ると誤解しやすい。海外投資家の現物売りと先物買いなど、合成ポジションとして解釈する。
- 金額ベース、枚数ベース、デルタ換算で意味が変わる。オプションを含む場合はデルタ換算の有無を明記する。

## 5. SQと限月効果
日経225先物・オプションでは、SQ（特別清算指数）に向けてヘッジ、ロール、裁定解消が集中しやすい。

- **メジャーSQ**: 3月、6月、9月、12月。先物とオプションの清算が重なる。
- **マイナーSQ**: それ以外の月。主にオプション清算。
- **ロール**: 期近限月から次限月へ建玉を移す取引。単純な売り買い方向だけでなく、限月間スプレッドを見る。
- **イベントリスク**: SQ週に米CPI、FOMC、日銀会合、雇用統計が重なると、IVとガンマの影響が増幅しやすい。
- **ピン・リスク (Pin Risk)**: SQ日に価格が建玉の集中するストライク付近に吸着される現象。ストライクをまたいで価格が行き来すると、デルタが0と1の間を激しく変動し、ヘッジコストが急増する。
  - 建玉の多いストライクは Max Pain・Gamma Wall と重なることが多く、相互に吸着効果を強める。
  - SQ日の後場はピンが外れると急速に離れることもあるため、ATM近辺のオプション保有者はSQ直前の対応を事前に決めておく。
