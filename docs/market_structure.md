# 市場構造と高度な分析手法 (Market Structure & Advanced Analysis)

日本市場、特に先物・オプション市場においてプロの投資家が注目する高度な分析概念です。

## 1. 建玉構造 (Open Interest Structure)
建玉（未決済残高）の分布は、将来の買い・売りの「潜在的な壁」や「加速要因」を示唆します。
- **最大マックスペイン (Max Pain)**: オプションの買い手の損失が最大、売り手の利益が最大となる価格帯。SQ（特別清算指数）に向けて価格が収束する傾向があるという仮説。
- **権利行使価格別の建玉分布**: 特定の価格帯（例：40,000円）に建玉が集中している場合、その価格が強い抵抗線または支持線として機能することがあります。

## 2. ガンマエクスポージャー (GEX)
マーケットメイカー（オプションの売り手）のヘッジ行動が原資産（日経平均）に与える影響を数値化した指標。
- **ポジティブ・ガンマ (Long Gamma)**: 相場のボラティリティを抑制する方向にヘッジが入る状態。
- **ネガティブ・ガンマ (Short Gamma)**: 相場が動いた方向にさらに加速させるヘッジ（追随売り・追随買い）が入りやすく、ボラティリティが拡大する状態。
- **ガンマ・フリップ (Gamma Flip)**: ガンマが正から負に転換する価格帯。ここを超えると相場が荒れやすくなります。

## 3. 裁定取引 (Arbitrage) とプログラム売買
現物株と先物の価格差を利用した取引。
- **裁定買い残**: 先物を売って現物を買っている状態。この残高が積み上がると、将来的に現物を売って先物を買い戻す「裁定解消売り」が発生し、株価の下げ要因となります。
- **逆裁定（裁定売り残）**: 将来的な買い戻し要因となります。

## 4. 投資家部門別売買動向
誰が日本株を動かしているかを把握するためのデータ。
- **海外投資家**: 日本株の売買シェアの約7割を占め、トレンドを形成。
- **個人投資家**: 逆張り（株価下落時に買い）の傾向が強い。
- **信託銀行**: 年金基金等のリバランス売買。
- **自己（証券会社）**: 裁定取引やマーケットメイク。
