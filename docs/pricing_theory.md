# プライシング理論 (Pricing Theory)

デリバティブ（金融派生商品）の適正価格を決定するための理論的枠組み。

## 1. 無裁定原理 (No-Arbitrage Principle)
「市場には『ただ飯（リスクなしで利益を得る機会）』は存在しない」という仮定。
- **一物一価の法則**: 同じキャッシュフローを生む商品は同じ価格でなければならない。
- **複製ポートフォリオ**: デリバティブと同じ支払いを実現する既存資産の組み合わせを作ることができれば、そのデリバティブの価格は複製コストと一致する。

## 2. リスク中立確率 (Risk-Neutral Probability)
将来の期待値を無リスク金利で割り引いた値が現時点の価格と一致するように定義された、理論上の確率測度。
- **核となる考え方**: リスク中立確率の下で将来のペイオフの期待値を算出し、それを無リスク金利で割り引くことで現在の適正価格を求めることができる。
- 実際の投資家が持つリスク選好（リスク回避的であるなど）を排除し、すべての資産がリスクフリーレートと同じ期待リターンを持つと仮定した仮想的な世界での確率分布。

## 3. ブラック・ショールズ・モデル (Black-Scholes Model)
欧州型オプションの価格を算出する代表的な数理モデル。
- **仮定**:
    - 株価は幾何ブラウン運動に従う。
    - リスクフリーレート、ボラティリティは一定。
    - 取引コストや税金がない。
- **計算式**: 株価、行使価格、残存期間、無リスク金利、ボラティリティの5つの変数で決定。

### 配当・指数先物を使う場合
日本株指数オプションでは、現物指数そのものではなく先物価格や配当落ちを意識する必要がある。

- **配当利回りを含む近似**: 連続配当利回り $q$ を使う場合、フォワード価格は $F = S e^{(r-q)T}$。
- **先物を原資産にする近似**: 先物価格 $F$ を使うBlack-76型の考え方では、配当や金利の影響が先物価格に反映される。
- **実務上の確認**: SQまでの配当落ち、限月間スプレッド、祝日を含む残存日数、清算値と理論価格の差を確認する。

### プット・コール・パリティ
同一限月・同一権利行使価格の欧州型コールとプットは、無裁定条件により原資産または先物価格と結び付く。

- **現物・配当利回りベース**: $C - P = S e^{-qT} - K e^{-rT}$
- **先物ベース**: $C - P = (F - K)e^{-rT}$
- **ATM先物近辺**: $F \approx K$ なら $C \approx P$。コールとプットの価格差は、先物と権利行使価格の乖離を割り引いたものになる。
- **注意**: 単に $C - P = F e^{-rT}$ ではなく、同じ権利行使価格 $K$ の現在価値も差し引く。実務では金利、配当、満期、手数料、ビッド・アスク、アメリカン型の早期行使可能性を考慮する。

### インプライド・ボラティリティ
市場価格から逆算されるボラティリティ。理論価格を市場価格に一致させる入力値であり、将来の実現ボラティリティを保証するものではない。

- **IV水準**: 市場がどの程度の変動を価格に織り込んでいるか。
- **スキュー**: 権利行使価格ごとのIV差。下落ヘッジ需要やテールリスクの反映として使われる。
- **ターム構造**: 限月ごとのIV差。イベント、SQ、金融政策発表の影響を受ける。

### Volatility Smile / Skew
ブラック・ショールズでは同一限月のIVは権利行使価格に依存しない前提だが、実際の市場では行使価格ごとにIVが異なる。

- **Smile**: ATMよりもOTMコール・OTMプットのIVが高く、曲線が笑顔のような形になる状態。大きな上下変動やジャンプリスクの価格付けを反映する。
- **Skew**: 片側のIVが高い状態。株価指数では下落ヘッジ需要によりOTMプットのIVが高くなりやすい。
- **25デルタ・リスクリバーサル**: 25デルタCall IV - 25デルタPut IV。上下どちらのテールが高く価格付けされているかを見る。
- **バタフライ**: OTMの平均IVとATM IVの差。分布のテールの厚さを把握する。

### Term Structure
限月ごとのIV水準の形状。満期までの時間軸で、市場がどの期間の変動を高く見積もっているかを示す。

- **Contango**: 期近IVより期先IVが高い状態。平常時に見られやすく、長期の不確実性が高く価格付けされる。
- **Backwardation**: 期近IVが期先IVより高い状態。決算、SQ、CPI、FOMC、日銀会合、ショック発生時など、近いイベントリスクが強い場合に起きやすい。
- **イベントIV**: 特定イベントを含む限月だけIVが盛り上がることがある。単純に年率IVを比較せず、イベント日をまたぐかを確認する。
- **実務上の注意**: 日経225オプションではSQまでの日数、祝日、メジャーSQ/マイナーSQ、先物限月との関係で期近IVが歪みやすい。

## 4. 二項モデル (Binomial Model)
株価が一定期間後に「上昇」または「下落」の2つの状態のみをとると仮定したモデル。
- 計算が直感的で、アメリカン型オプション（期間中いつでも行使可能）の評価や、リスク中立確率による価格付けの理解に適している。

## 5. モンテカルロ・シミュレーション
リスク中立確率の下で将来の資産価格の経路を数万回シミュレーションし、その期待ペイオフを算出して割り引く手法。
- 複雑なペイオフを持つエキゾチック・オプションの評価に多用される。

## 6. モデル価格と市場価格の違い
モデル価格は「前提条件の下での整合価格」であり、市場で必ず約定できる価格ではない。

- **流動性**: 板が薄い銘柄や遠い権利行使価格では、理論価格よりもビッド・アスクスプレッドの方が重要になる。
- **モデルリスク**: 正規分布、一定ボラティリティ、連続ヘッジといった仮定は現実には成立しない。
- **データ品質**: 清算値、理論価格、直近約定、気配値は意味が異なる。分析目的に応じて優先順位を決める。
- **検証方法**: モデル価格だけでなく、過去の実現損益、ヘッジ誤差、スリッページ、イベント時の挙動で検証する。
