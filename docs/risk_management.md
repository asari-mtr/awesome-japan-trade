# リスク管理 (Risk Management)

予期せぬ損失から資産を守るための測定と制御の手法。

## 1. VaR (Value at Risk)
一定期間（保有期間）と確率（信頼水準）において、損失額がその値を超える確率が一定以下（1-信頼水準）になるような損失額の推定値（分位点）。
- 例：1日 VaR 99% = 100万円。これは「統計的に、1日で100万円を超える損失が発生する確率は1%以下である」ことを意味する。
- **注意**: VaRは「最大損失」そのものではなく、それを超える損失（テールイベント）の規模については言及しない。

## 2. 期待ショートフォール (ES: Expected Shortfall)
VaRを超える損失が発生した場合の平均的な損失額。
- テールリスク（VaRを超える極端な損失）の大きさを捉えることができ、VaRよりも保守的かつ一貫性のある（コヒーレントな）リスク指標とされる。

## 3. グリークス (Greeks / 指標)
ポートフォリオの価値が各変数に対してどれだけ変化するかを示す指標。主にオプション取引で使用される。
- **デルタ ($\Delta$)**: 原資産価格の変化に対するポートフォリオ価値の変化。
- **ガンマ ($\Gamma$)**: デルタの変化率（原資産価格の変化の2次微分）。
- **ベガ ($\nu$)**: 原資産のボラティリティの変化に対するポートフォリオ価値の変化。
- **セータ ($\theta$)**: 時間の経過（タイムディケイ）に対するポートフォリオ価値の変化。
- **ロー ($\rho$)**: 無リスク金利の変化に対するポートフォリオ価値の変化。

## 4. ストレス・テスト
過去の金融危機（リーマンショック等）や仮想的な最悪シナリオを想定し、ポートフォリオが耐えられるかをシミュレーションする。VaRのような統計モデルが機能しない異常事態のリスクを把握する。

## 5. 信用リスク管理
取引相手のデフォルト（債務不履行）に伴うリスクの管理。
- **PD (Probability of Default)**: デフォルト確率。
- **LGD (Loss Given Default)**: デフォルト時損失率。
- **EAD (Exposure at Default)**: デフォルト時与信額。
