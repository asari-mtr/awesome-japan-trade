# リスク管理 (Risk Management)

予期せぬ損失から資産を守るための測定と制御の手法。

## 1. VaR (Value at Risk)
一定期間（保有期間）と確率（信頼水準）において、発生しうる最大損失額。
- 例：1日 VaR 99% = 100万円。これは「100回に1回、1日で100万円以上の損失が出る可能性がある」ことを意味する。
- **欠点**: 信頼水準を超える（1%の確率で起きる）極端な損失（テールリスク）の大きさを把握できない。

## 2. 期待ショートフォール (ES: Expected Shortfall)
VaRを超える損失が発生した場合の平均損失額。
- テールリスクを捉えることができ、VaRよりも保守的なリスク指標とされる。

## 3. グリークス (Greeks / 指標)
ポートフォリオの価値が各変数に対してどれだけ変化するかを示す指標。主にオプション取引で使用される。
- **デルタ ($\Delta$)**: 原資産価格の変化に対するポートフォリオ価値の変化。
- **ガンマ ($\Gamma$)**: デルタの変化率（原資産価格の変化の2次微分）。
- **ベガ ($\nu$)**: ボラティリティの変化に対する変化。
- **セータ ($\theta$)**: 時間の経過（タイムディケイ）に対する変化。
- **ロー ($\rho$)**: 金利の変化に対する変化。

## 4. ストレス・テスト
過去の金融危機（リーマンショック等）や仮想的な最悪シナリオを想定し、ポートフォリオが耐えられるかをシミュレーションする。

## 5. 信用リスク管理
取引相手のデフォルト（債務不履行）に伴うリスクの管理。
- **PD (Probability of Default)**: デフォルト確率。
- **LGD (Loss Given Default)**: デフォルト時損失率。
- **EAD (Exposure at Default)**: デフォルト時与信額。
