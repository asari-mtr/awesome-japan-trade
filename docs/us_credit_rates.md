# 米国社債・短期金利・イールドカーブ

米国社債市場は、短期金利、米国債利回り、信用スプレッド、景気サイクルをまとめて見る必要がある。株式、為替、日本株にも波及しやすい。

## 1. 社債の基本

社債は企業が資金調達のために発行する債券。投資家は利息（クーポン）と元本返済を受け取る代わりに、企業の信用リスクを負う。

| 項目 | 内容 |
| :--- | :--- |
| **額面** | 満期に返済される元本 |
| **クーポン** | 定期的に支払われる利息 |
| **満期** | 元本返済日 |
| **利回り** | 現在価格に対する投資収益率 |
| **信用格付け** | 発行体の返済能力の評価 |
| **信用スプレッド** | 同年限の米国債利回りに対する上乗せ利回り |

社債利回りは大きく分けると次の形で考える。

$$y_{\text{corp}} \approx y_{\text{UST}} + s_{\text{credit}}$$

米国債利回りは金利リスク、信用スプレッドは企業信用リスクと流動性リスクを反映する。

## 2. 投資適格債とハイイールド債

| 区分 | 特徴 | 市場での読み方 |
| :--- | :--- | :--- |
| **Investment Grade (IG)** | 信用力が高い社債 | 金利感応度が大きく、米国債利回りの影響を受けやすい |
| **High Yield (HY)** | 信用力が低い社債 | 株式に近く、景気・デフォルト懸念・流動性の影響を受けやすい |

IGは金利商品としての性格が強く、HYは信用リスク資産としての性格が強い。

## 3. 短期金利との関係

短期金利はFRBの政策金利、SOFR、T-Bill利回りなどに反映される。短期金利が上がると、企業の借換コストや変動金利負債の負担が上がりやすい。

- **短期金利上昇**: 借換コスト上昇、企業利益圧迫、ハイイールド債に逆風。
- **短期金利高止まり**: 満期到来企業のリファイナンスリスクが増える。
- **短期金利低下**: 借換環境は改善しやすいが、景気悪化による利下げなら信用スプレッドは拡大し得る。
- **信用スプレッドとの組み合わせ**: 金利低下でも、スプレッド拡大が大きければ社債価格は下がることがある。

## 4. 社債スプレッドが市場に与える影響

社債スプレッドは、企業の資金調達環境とリスク許容度を示す重要な指標。

| スプレッドの動き | 解釈 | 影響 |
| :--- | :--- | :--- |
| **縮小** | 信用不安が低い、リスクオン | 株高、HY債高、円安・高β資産に追い風になりやすい |
| **拡大** | 信用不安、景気後退懸念、流動性悪化 | 株安、HY債安、ドル資金需要増、円高圧力になりやすい |
| **急拡大** | クレジットイベントや金融ストレス | ボラティリティ上昇、流動性低下、リスク資産の連鎖売り |

社債スプレッド拡大は、株式市場の下落に先行または同時進行することがある。特にHYスプレッドは景気敏感で、米小型株や高レバレッジ企業の動きと連動しやすい。

## 5. イールドカーブ

イールドカーブは、米国債の年限ごとの利回りを並べた曲線。景気、インフレ、政策金利、タームプレミアムを反映する。

| 形状 | 内容 | 市場での読み方 |
| :--- | :--- | :--- |
| **順イールド** | 長期金利 > 短期金利 | 通常の景気拡大局面で見られやすい |
| **フラット化** | 長短金利差が縮小 | 利上げ局面、将来成長期待の低下 |
| **逆イールド** | 短期金利 > 長期金利 | 引き締め強化、将来利下げ・景気減速の織り込み |
| **スティープ化** | 長短金利差が拡大 | 利下げ期待、財政懸念、成長/インフレ期待の変化 |

## 6. 10Y-2Y金利差

10年国債利回り - 2年国債利回りは、代表的な長短金利差。

$$\text{10Y--2Y} = y_{10Y} - y_{2Y}$$

- **低下/逆イールド化**: FRBの引き締めや将来景気減速の織り込みを示しやすい。
- **ブル・スティープ化**: 短期金利が大きく低下してカーブが立つ。利下げ期待や景気悪化懸念を伴うことが多い。
- **ベア・スティープ化**: 長期金利が大きく上昇してカーブが立つ。財政懸念、タームプレミアム上昇、インフレ懸念を伴うことがある。

逆イールドは「すぐに株が下がる」シグナルではない。重要なのは、逆イールドの発生そのものより、その後のスティープ化が景気悪化によるものか、長期金利上昇によるものかを分けること。

## 7. 日本株への波及

- **米長期金利上昇**: グロース株、半導体株のバリュエーションに逆風。
- **短期金利高止まり**: 米景気と信用市場に負担。HYスプレッド拡大ならリスクオフになりやすい。
- **信用スプレッド拡大**: 海外投資家のリスク許容度低下を通じて日本株売りにつながることがある。
- **ドル円経由**: 金利差は円安/円高に影響し、輸出株・内需株の相対パフォーマンスを動かす。
